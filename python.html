Assignments
  • Developing and testing your Advanced Calculator using python
    1. There should be a GUI to operate and display Final Answers
    2. It should include Basic operators ( +,-,*,/)
    3. It should include Advance options like sin, cos , tan etc..
    4. There must be a proper validation
    5. There should be last 10 recall option to verify the entries
  • Develop customized Share Market Price Tracker : Phase 1
    1. Develop a GUI to take a input of the company Name
    2. Use Google API to interact with Google search engine.
    3. Fetch the Result and Display all the interested parameters
    4. Must have proper validation, and Results should be accurate
    5. Display the price graph , provide option for 1 year , 5 year , max
  • Develop customized Share Market Price Tracker : Phase 2
    1. Using the existing tool from phase 1 to Enhance Additional Requirement
    2. Provide a option to have a watch list of companies.
    3. Max 5 Watch list should be provided , Each can hold up to 50 companies
    4. Display the current Price , Low price and High price
    5. Accurate Data of 1 year low and high price should be displayed
  • Customize Data scraping using python
    1. Scrap the data from the weather forecast website
    2. Save all the data in the Excel sheet for analysis
    3. Provide the option to show the past year data
    4. Provide the option to display the different types of chart , Drop down menu's for different country etc.
